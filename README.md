# Skills Repository

This repository stores AI skill definitions and related configuration for local automation and security workflows.

## Included skill

- `local-security-audit/` – a local security audit skill designed to analyze and remediate vulnerabilities in a localhost development environment.

## Usage

1. Open the `local-security-audit` folder to inspect the skill manifest and prompt content.
2. Keep skill metadata aligned with the folder name for consistent loading.


## Notes

- This repo is intended for local AI skill development and should not be used for external scanning or public vulnerability disclosure.
